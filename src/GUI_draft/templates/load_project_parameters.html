Load Project Parameters View-Component

  • This view component is rendered as a button with the text 'Load Parameters from file...'
  • It allows the user to load an existing project file, pre-defined or already-saved from previous simulations
  • Uploading the file lets the program automatically fill-in the required project parameters
  • The corresponding view-component is yet to be described in readthedocs
  • It is connected to the view-component to read_write_files (file explorer)
